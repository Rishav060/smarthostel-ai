SmartHostel AI

Autonomous Hostel Operations, Safety & Wellbeing Platform

SmartHostel AI is an AI-powered hostel management platform designed to improve hostel operations, student safety, emergency response, and overall student wellbeing.

The platform provides dedicated portals for Students, Parents, Wardens, Security Staff, and Administrators, enabling efficient communication and management through a unified system.

---

Problem Statement

Traditional hostel management relies on manual processes, paper registers, and scattered communication channels, making it difficult to manage:

- Complaints
- Room allocations
- Visitor management
- Leave requests
- Emergency situations
- Student-parent communication
- Security operations

SmartHostel AI addresses these challenges through a centralized and intelligent platform.

How to Run

Option 1: Open Directly

1. Download the repository.
2. Open "index.html" in any modern browser.

Option 2: Local Server

Using VS Code Live Server:

1. Open the project folder in VS Code.
2. Install the Live Server extension.
3. Right-click "index.html".
4. Click Open with Live Server.
